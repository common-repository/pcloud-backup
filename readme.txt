pCloud Backup va vous permettre de réaliser des sauvegardes de votre/vos site(s) WordPress (et WooCommerce) directement sur votre cloud pCloud.

== Description ==

**Vous allez pouvoir / You will be able to :**

Réaliser des sauvegardes instantanées,
Sélectionner depuis votre back office WordPress le dossier dans lequel vous souhaitez sauvegarder votre WordPress.
Choisir si vous souhaitez sauvegarder uniquement la base de données, les fichiers ou les deux.

Create instant backups of your website,
Select from your WordPress the folder where you want to save your backup.
Select if you want to save only the database, files and folders or both.

**Comment faire ? / How ? :**

Une fois l’extension activée, il suffit de vous rendre dans l’onglet API et de renseigner votre API pCloud.

Once the plugin activate, just go to the API tab and enter your pCloud API 

**<a href="https://www.josselynjayant.fr/pcloud-backup-wordpress-plugin/">Disponible en version pro !</a>**

Programmer des sauvegardes tous les jours, toutes les semaines ou tous les mois en plus des fonctionnalités de la version gratuite !

**<a href="https://www.josselynjayant.fr/pcloud-backup-wordpress-plugin/">Pro version available !</a>**

With pro version you can scheduled backup everyday, every week or every month.
No limitation on number of installations !

== Frequently Asked Questions ==

= Comment faire pour programmer des sauvegardes ? =

La version gratuite ne permet pas de faire cela. Pour programmer des sauvegardes, vous devez utiliser la version pro de pCloud Backup.


= Comment récupérer mes identifiants API ? How to get my pCloud API credentials ? =

Il suffit de vous rendre ici : https://docs.pcloud.com/my_apps/
Go to this URL : https://docs.pcloud.com/my_apps/


= Dois-je attendre la fin de la sauvegarde avant de fermer ? =

Avec la version gratuite de l'extension, lors de la création d'une sauvegarde instantanée, vous devez avoir la fenêtre de chargement du fichier ouverte tant que la notification vous indiquant que la sauvegarde est terminée n'est pas affichée.
